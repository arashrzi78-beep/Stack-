📦 Stack Implementation in Java

A simple Stack data structure implementation in Java using an array-based approach.
This project demonstrates fundamental data structures and object-oriented programming concepts such as encapsulation, stack operations, and class interaction.

🚀 Features

📥 Push – Add elements to the top of the stack

📤 Pop – Remove the top element from the stack

👀 Peek – View the top element without removing it

📊 Size – Get the number of elements in the stack

❌ isEmpty() – Check if the stack is empty

⚠️ isFull() – Check if the stack is full

The program also demonstrates:

Generating random numbers

Storing them inside a stack

Printing stack contents

Using a temporary stack to preserve order

🧠 Concepts Demonstrated

This project practices core programming and computer science concepts:

Data Structures (Stack – LIFO principle)

Object-Oriented Programming (OOP)

Encapsulation

Array-based storage

Stack traversal using temporary structures

Random number generation in Java

⚙️ How It Works

A stack with a capacity of 10 elements is created.

The program generates random numbers and pushes them into the stack.

The stack elements are printed using a temporary stack to avoid losing the original order.

The stack operations demonstrate how LIFO structures behave.

💻 Example Stack Operations
stack.push(10);
stack.push(20);
stack.push(30);

System.out.println(stack.peek()); // 30

stack.pop(); // removes 30
